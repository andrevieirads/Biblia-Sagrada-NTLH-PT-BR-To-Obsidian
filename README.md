# Bíblia Sagrada para o Obsidian

**Versão:** NTLH (Português - Brasil)

---

## Sobre mim

Prazer, me chamo André. Sou cristão e católico. Este projeto não é sobre mim, mas sim sobre glorificar a Deus e tornar Sua Palavra mais acessível.

## Objetivo do projeto

O objetivo deste projeto é disponibilizar a Palavra de Deus em formato Markdown, com links autorreferenciáveis, para ser utilizada no Obsidian. A proposta é facilitar o estudo bíblico, permitindo que cristãos aprofundem seu conhecimento por meio de uma leitura mais conectada, organizada e interativa dentro da ferramenta.

## Observações importantes

Sou católico, porém a versão desta Bíblia é baseada no cânon protestante.

Isso ocorre porque a API utilizada para gerar o conteúdo disponibiliza apenas versões protestantes da Bíblia. Por esse motivo, os 7 livros deuterocanônicos da tradição católica não estão incluídos:

- Tobias
- Judite
- Sabedoria
- Eclesiástico (Sirácida)
- Baruc
- 1 Macabeus
- 2 Macabeus

Caso no futuro seja possível obter uma fonte confiável com esses livros, pretendo incluí-los no projeto.

## Como usar?

Siga os passos abaixo para acessar a Bíblia no Obsidian:

### 1. Instalar o Obsidian

O Obsidian é um aplicativo gratuito usado para organizar e ler notas em formato Markdown.

1. Acesse o site oficial: [https://obsidian.md](https://obsidian.md/)
2. Clique em “Download”
3. Instale o aplicativo normalmente no seu computador (como qualquer outro programa)

### 2. Baixar o conteúdo da Bíblia

Você não precisa saber usar GitHub para isso. Basta baixar o projeto como um arquivo compactado:

1. Acesse o repositório:  
    [https://github.com/andrevieirads/Biblia-Sagrada-NTLH-PT-BR-To-Obsidian](https://github.com/andrevieirads/Biblia-Sagrada-NTLH-PT-BR-To-Obsidian)
2. Clique no botão verde escrito “Code”
3. Clique em “Download ZIP”
4. Após o download, extraia o arquivo ZIP em uma pasta no seu computador  
    (por exemplo: na Área de Trabalho ou em Documentos)

### 3. Abrir o projeto no Obsidian

Agora você vai abrir essa pasta dentro do Obsidian:

1. Abra o Obsidian
2. Clique em “Open folder as vault” (ou “Abrir pasta como cofre”)
3. Selecione a pasta que você extraiu no passo anterior
4. Clique em “Open” (ou “Abrir”)

### 4. Começar a leitura

Após abrir a pasta:

- Você verá os livros da Bíblia organizados em notas
- Clique em qualquer arquivo para começar a leitura
- Os links dentro dos textos permitem navegar entre capítulos e referências

### Observação

O Obsidian funciona como um sistema de anotações interligadas. Com o tempo, você pode:

- Criar suas próprias anotações
- Destacar trechos importantes
- Fazer conexões entre versículos
